# COINFIN 🎣

**Catch the Ecosystem** — A premium mobile-first browser fishing game where every fish represents a coin from the Orynth ecosystem.

## Features

- 🌊 7-layer living ocean canvas renderer
- 🐟 8 fish species across 5 rarity tiers (Common → Legendary)
- 🎮 Full fishing state machine (cast → sink → wait → attract → bite → reel → catch)
- 📖 Fishpedia & Collection tracker
- 🌙 Day/night cycle with dynamic weather
- 🎵 Procedural Web Audio engine
- ⭐ XP / level progression & 5 zone unlock system
- 💾 Auto-save via localStorage

## Play

Open `index.html` directly in your browser — or visit the live deployment.
